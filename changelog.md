# Changelog

Усі помітні зміни в проекті CRM AI Consultant будуть документуватися в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), і цей проєкт дотримується [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.7.0] - 2026-04-07

### ✨ Додано

- Повноцінну сторінку `conversations.php` з таблицею історії

- Кращу адаптивність таблиці розмов на мобільних пристроях
- Коротке відображення повідомлень у таблиці

### 🔧 Змінено
- Оновлено шлях до папки `conversations/` (тепер `/ai/crm/conversations/`)
- Покращено витягнення `site_id` з назв файлів
- Оновлено дизайн таблиці історії розмов

### 🛠 Виправлено
- Помилки підключення `functions.php` в адмін-панелі
- Проблеми з відображенням довгих повідомлень у історії

## 🎉 Основні зміни

### ✅ Виправлено
- Критична помилка з MIME-type (`text/plain` замість `application/javascript`) — віджет тепер стабільно завантажується на всіх сайтах
- Помилка "Функція Telegram не знайдена" — всі канали тепер підключаються явно
- Uncaught SyntaxError: string literal contains an unescaped line break
- Проблеми з CORS при AJAX-запитах
- Неправильне підключення файлів налаштувань (`settings_*.php`)

### ✨ Додано
- **Повністю робоча інтеграція Grok (xAI)** з системним промптом, вибором моделі та температурою
- **Повністю робоча інтеграція OpenAI (ChatGPT)** з системним промптом, вибором моделі та температурою
- Розширені налаштування для **WhatsApp** (номер, привітання, автовідповідь, тип акаунту)
- Розширені налаштування для **Viber** (номер, привітання, автовідповідь, статус)
- Перемикач "Увімкнено / Вимкнено" чату в налаштуваннях сайту
- Оновлена шапка адмін-панелі з посиланням на "Історію чатів"
- Детальне логування помилок для кожного каналу

### 🔧 Технічні покращення
- Всі канали тепер підключаються явно в `functions.php`
- Покращена обробка помилок з використанням `Throwable`
- Оптимізовано вивід JavaScript у `index.php` (більш сумісний з CSP)
- Додано примусове очищення буфера виводу
- Покращено збереження та завантаження налаштувань сайтів

### 📋 Для користувача
- Можливість окремо налаштовувати системний промпт для OpenAI та Grok
- Прямі посилання на чат у WhatsApp (`wa.me`)
- Краща видимість поточного каналу та моделі в адмін-панелі
- Стабільніша робота віджету на сайтах з жорстким Content Security Policy
