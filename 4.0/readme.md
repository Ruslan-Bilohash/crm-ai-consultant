# CRM AI Consultant

**Універсальна система AI-чату** з підтримкою кількох каналів і повними індивідуальними налаштуваннями для кожного сайту.

Підтримує: **Telegram • Grok (xAI) • OpenAI (GPT) • WhatsApp • Viber**

---

## ✨ Що нового в версії 4.0 (2026)

### Головні зміни
- **Повний перехід на MySQL** — більше немає JSON-файлів для сайтів і історії чатів
- Авторизація адміністратора через базу даних з безпечним хешуванням паролів (Argon2ID)
- **Окремі системні промпти для кожного каналу**:
  - `grok_system_prompt`
  - `openai_system_prompt`
  - `telegram_system_prompt`, `whatsapp_system_prompt`, `viber_system_prompt`
- Повернено **всі налаштування дизайну чату** (кольори віджету, бульбашок, хедера, позиція тощо)
- Нова сторінка **Управління адміністратором** (`admins.php`) для зміни пароля
- Сучасний світлий дизайн шапки адмін-панелі
- Виправлено всі помилки з `session_start()` та дублюванням констант

### Технічні покращення
- Надійна робота з PDO
- Краща перевірка підключення віджету на сайті клієнта
- Покращена безпека та обробка помилок
- Адаптивний дизайн адмін-панелі

---

## 🚀 Основні можливості

- Підключення AI-чату на **будь-який сайт** (WordPress, Tilda, HTML, Laravel тощо)
- Повністю **індивідуальні налаштування** для кожного сайту
- 5 каналів зв’язку:
  - Telegram (найшвидший)
  - Grok (xAI)
  - OpenAI (GPT-4o, o1 тощо)
  - WhatsApp (пряме посилання)
  - Viber (пряме посилання)
- Індивідуальний дизайн чату (кольори, іконка, позиція, заголовок)
- Історія всіх розмов
- Перевірка наявності коду віджету на сайті

---

## Встановлення

1. Завантажте файли в папку на сервері (наприклад `/ai/crm/`)
2. Налаштуйте підключення до бази даних у файлі `config.php`
3. Виконайте SQL для створення таблиць (`admins` та `sites`)
6. Зайдіть у адмін-панель: `https://your-site.com/ai/crm/admin/login.php`

**За замовчуванням:**
- Логін: `admin`
- Пароль: `12345` (обов’язково змініть після першого входу!)
