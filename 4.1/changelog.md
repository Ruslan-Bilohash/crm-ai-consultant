# Changelog

Всі значні зміни в проєкті CRM AI Consultant.

## [4.1] - 2026-04-25

**Major Performance Update**

### Додано
- **Промпт тепер зберігається у файл** (`sites/{site_id}_prompt.txt`) — значно швидші відповіді AI
- **Кешування налаштувань сайту** у JSON-файли
- **Опція "Надсилати також у Telegram"** — дублювання повідомлень навіть при Grok/OpenAI
- Новий блок Changelog у документації
- Окремий файл `changelog.html`

### Оптимізовано
- Обробники `grok.php` та `openai.php` (менші таймаути, швидший cURL)
- Зменшено навантаження на базу даних під час чату
- Покращена стабільність кешу

### Змінено
- Головний промпт тепер використовує поле `system_prompt`
- Оновлено документацію та дизайн

## [4.0] - 2026-04-21

**Велике оновлення**

### Додано
- Повний перехід на **MySQL** (замість JSON-файлів)
- Сучасний дизайн адмін-панелі
- Підтримка 5 каналів: Telegram, Grok (xAI), OpenAI, WhatsApp, Viber
- Індивідуальні налаштування для кожного сайту (дизайн, промпт, канал)
- Історія чатів у базі даних
- Вибір іконок бота
- Перевірка підключення віджету

### Змінено
- Архітектура проєкту
- Структура папок (`sites/cache/`, файловий промпт)
