Потужний AI-віджет з підтримкою **Grok (xAI), OpenAI, Telegram, WhatsApp та Viber**. Повний контроль, швидкі відповіді та індивідуальний дизайн.

![CRM AI Consultant](screen/wigets.jpg)

## ✨ Основні можливості

- 5 каналів в одному скрипті
- Дуже швидкі відповіді AI (промпт у файлі)
- Повністю індивідуальні налаштування для кожного сайту
- Історія чатів у MySQL
- Адаптивний сучасний дизайн
- 100% самохостинг

## Встановлення

1. Завантажте файли після оплати
2. Налаштуйте `config.php` (база даних)
3. Створіть папки `sites/` та `sites/cache/` (права 755)
4. Зайдіть в `/admin/` (admin / 12345)
