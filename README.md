# crm-ai-consultant
CRM AI Consultant — це універсальна система двостороннього AI-чату, яка дозволяє підключати віджет на будь-який сайт. Кожен сайт має повністю індивідуальні налаштування: дизайн, канал зв'язку, модель AI, API ключі тощо.  Підтримувані канали: Telegram, OpenAI (GPT), Grok (xAI), WhatsApp, Viber
