# 🤖 CRM AI Consultant

**Універсальна система AI-чату** з індивідуальними налаштуваннями для кожного сайту.

Підтримує кілька каналів зв'язку, повну історію чатів та зручну адмін-панель.

---

## ✨ Основні можливості

- Підключення віджету на **будь-який сайт** одним рядком коду
- Повністю незалежні налаштування для кожного сайту
- Підтримка каналів: **Telegram, OpenAI (GPT), Grok (xAI), WhatsApp, Viber**
- Індивідуальний дизайн чату (кольори, іконка, позиція, привітання)
- Повна історія спілкування в адмін-панелі
- Автоматична генерація `site_id` з домену (наприклад: `mapsme_no`)
- Перевірка наявності коду на сайті
- Адаптивний сучасний інтерфейс

Як використовувати

Додайте новий сайт в адмін-панелі
Заповніть налаштування (канал, API ключі, дизайн)
Скопіюйте згенерований код
Вставте його на свій сайт перед </body>:

Підтримувані канали

Telegram — найшвидший канал
OpenAI (GPT) — автоматичні відповіді
Grok (xAI) — автоматичні відповіді
WhatsApp — в розробці
Viber — в розробці


API ключі

Telegram: Bot Token + Chat ID
OpenAI: platform.openai.com/api-keys
Grok: x.ai/api

Вирішення проблем

Білий екран → Перевірте файл crm-ai-error.log
Чат не з’являється → Перевірте правильність site_id та місце вставки скрипту
Помилка зв'язку → Перевірте налаштування каналу та API ключі
