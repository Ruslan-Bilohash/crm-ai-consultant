# 🤖 CRM AI Consultant

**Універсальна система AI-чату** з індивідуальними налаштуваннями для кожного сайту.

Підтримує кілька каналів зв'язку, повну історію чатів та зручну адмін-панель.

---

## ✨ Основні можливості

- Підключення віджету на **будь-який сайт** одним рядком коду
- Повністю незалежні налаштування для кожного сайту
- Підтримка каналів: **Telegram, OpenAI (GPT), Grok (xAI), WhatsApp, Viber**
- Індивідуальний дизайн чату (кольори, іконка, позиція, привітання)
- Повна історія спілкування в адмін-панелі
- Автоматична генерація `site_id` з домену (наприклад: `mapsme_no`)
- Перевірка наявності коду на сайті
- Адаптивний сучасний інтерфейс
