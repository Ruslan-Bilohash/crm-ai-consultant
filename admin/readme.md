
# Адмін-панель CRM AI Consultant

Цей документ описує всі можливості адмін-панелі.

Адмін-панель знаходиться за адресою:  
**`/ai/crm/admin/`**

---

## Головні сторінки адмін-панелі

| Сторінка                    | Файл                          | Опис |
|----------------------------|-------------------------------|------|
| **Головна**                | `admin/index.php`             | Список всіх сайтів |
| **Налаштування сайту**     | `admin/sites.php`             | Повне налаштування одного сайту |
| **Історія розмов**         | `admin/conversations.php`     | Список усіх чатів |
| **Перегляд розмови**       | `admin/get-conversation.php`  | AJAX-завантаження розмови в модальне вікно |

---

## Детальний опис функцій

### 1. Список сайтів (`admin/index.php`)
- Показує всі додані сайти
- Статус чату (● Увімкнено / ○ Вимкнено)
- Кількість розмов по кожному сайту
- Кнопки: **Редагувати** та **Видалити**

### 2. Налаштування сайту (`admin/sites.php`)

Це основний і найважливіший файл адмін-панелі.

#### Доступні налаштування:

**Загальні:**
- Назва сайту
- Домен
- Опис сайту
- Статус чату (увімкнути / вимкнути)

**Канал за замовчуванням:**
- Telegram
- OpenAI (ChatGPT)
- Grok (xAI)
- WhatsApp
- Viber

**Налаштування чату:**
- Заголовок чату
- Підзаголовок чату
- Іконка бота
- **Привітання користувача** (перше повідомлення)
- **Автоматичне відкриття чату** + затримка в мілісекундах
- **Максимальна кількість повідомлень в історії**

**Дизайн чату:**
- Колір кнопки віджету
- Колір фону чату
- Колір хедера чату
- Колір бульбашки користувача
- Колір бульбашки бота
- Позиція віджету (ліва / права сторона)

**Канали:**
- **Telegram** — токен та Chat ID
- **OpenAI** — API ключ + System Prompt
- **Grok (xAI)** — API ключ + System Prompt
- **WhatsApp** — номер телефону
- **Viber** — номер телефону

---

### 3. Історія розмов (`admin/conversations.php`)
- Показує всі розмови, відсортовані за часом (нові зверху)
- Відображає:
  - Назву сайту
  - Кількість повідомлень
  - Останнє повідомлення
  - Час останнього повідомлення
- Клік по рядку або кнопка «Переглянути» відкриває повну розмову

### 4. Перегляд розмови (модальне вікно)
- Завантажується через AJAX (без перезавантаження сторінки)
- Показує всю історію чату
- Повідомлення клієнта та бота виділені різними кольорами
- Показує час кожного повідомлення

---

## Як користуватися адмін-панеллю

1. Зайди в `/ai/crm/admin/`
2. Увійди (пароль за замовчуванням: `12345`)
3. Додай новий сайт або відредагуй існуючий
4. Налаштуй канал, дизайн і поведінку чату
5. Скопіюй скрипт і встав його на свій сайт
6. Переглядай історію розмов у розділі «Історія розмов»

---

**Порада:**  
Після кожного збереження налаштувань обов’язково оновлюй сторінку з жорстким кешем (`Ctrl + Shift + R`), щоб зміни застосувалися.
